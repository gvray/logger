📡 API Request Logging

HTTP 请求日志 常用

记录 API 请求的完整生命周期：请求参数、响应时间、状态码、错误信息等。 使用命名空间区分不同的 API 模块，使用中间件添加请求 ID。 

// 配置
const apiLogger = logger.child('api');
apiLogger.use(prefixMiddleware(() => `[${generateRequestId()}]`));

// 使用
apiLogger.info('GET /users', { params: { page: 1 } });
apiLogger.debug('Response:', { status: 200, time: '45ms' });

🐛 Error Handling & Debugging

错误追踪与调试 重要

捕获和记录应用错误，包括堆栈信息、上下文数据、用户操作等。 使用监听器将错误发送到监控服务（如 Sentry）。 

// 错误处理
try {
  await riskyOperation();
} catch (error) {
  logger.error('Operation failed:', error, {
    userId: currentUser.id,
    action: 'checkout',
    cart: cartItems
  });
}

⚡ Performance Monitoring

性能监控 优化

记录关键操作的性能指标：执行时间、内存使用、资源加载等。 使用自定义中间件计算时间差，监听器收集性能数据。 

// 性能测量
const start = performance.now();
await heavyOperation();
const duration = performance.now() - start;

perfLogger.info('Operation completed', {
  duration: `${duration.toFixed(2)}ms`,
  memory: performance.memory?.usedJSHeapSize
});

👤 User Activity Tracking

用户行为追踪

记录用户在应用中的关键操作：登录、页面访问、功能使用等。 使用命名空间区分不同用户，中间件自动添加用户信息。 

// 用户日志
const userLogger = logger.child(`user:${userId}`);
userLogger.use((ctx, next) => {
  ctx.args = [...ctx.args, { sessionId, timestamp }];
  next();
});

userLogger.info('Page viewed', { page: '/dashboard' });
userLogger.info('Button clicked', { button: 'checkout' });

🔄 State Management

状态变更日志

记录应用状态的变化，便于调试和回溯。 特别适合 Redux、Vuex 等状态管理库的集成。 

// Redux middleware
const loggerMiddleware = store => next => action => {
  stateLogger.debug('Action:', action.type, {
    payload: action.payload,
    prevState: store.getState()
  });
  const result = next(action);
  stateLogger.debug('New state:', store.getState());
  return result;
};

🔐 Security & Audit

安全审计日志 安全

记录安全相关事件：登录尝试、权限检查、敏感操作等。 使用脱敏中间件保护敏感信息，监听器发送到审计系统。 

// 安全日志
const auditLogger = logger.child('audit');
auditLogger.use(redactMiddleware([
  /password/i,
  /token/i,
  /secret/i
]));

auditLogger.warning('Failed login attempt', {
  username: 'admin',
  ip: '192.168.1.100',
  reason: 'invalid_password'
});
